Description
============

This folder contains Indonesian POS tagging dataset that is used in:

Kurniawan, Kemal, and Alham Fikri Aji. 2018. Toward a Standardized and
More Accurate Indonesian Part-of-Speech Tagging. ArXiv:1809.03391 [Cs].
http://arxiv.org/abs/1809.03391.

which is just a cross-validation split of the original dataset described
in:

Dinakaramani, Arawinda, Fam Rashel, Andry Luthfi, and Ruli Manurung. 2014.
Designing an Indonesian Part of Speech Tagset and Manually Tagged Indonesian
Corpus. In 2014 International Conference on Asian Language Processing (IALP)
Pp. 66â€“69. IEEE. http://ieeexplore.ieee.org/document/6973519/.

There are training, development, and test set for the 5 cross-validation
folds. Their filenames adhere to the following format:

* train.01.tsv - the training set for fold 1
* dev.01.tsv - the development set for fold 1
* test.01.tsv - the test set for fold 1

Filenames for fold 2-5 follow the same format as above.
